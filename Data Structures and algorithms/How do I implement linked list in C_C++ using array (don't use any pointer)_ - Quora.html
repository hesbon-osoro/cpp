The idea behind creating having linked lists is to create nodes on the fly and link them to their preceding and/or succeeding nodes unlike arrays.
In arrays the memory is alloted during compile time and it is static while what you would need for a linked list is dynamic allocation of memory during run time(which can be done using pointers in C/C++).
Though what you have asked for is actually doable , it kind of overshadows the purpose of having a linked list.
